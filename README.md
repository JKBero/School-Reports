# School Reports

## Question

A school inputs their exam results and a report is produced showing the total number of each grade (Green, Amber or Red).

## Initial Planning

| Inputs | Outputs |
| ------ | ------- |
| 'Green' | "Green: 1" |
| 'Amber' | "Amber: 1" |
| 'Red' | "Red: 1" |
| 'Green, Green' | "Green: 2" |
| 'Green, Amber' | "Green: 1\nAmber: 1" |
